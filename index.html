Cómo empezar

Sigue este flujo para elegir un modelo, ejecutar los prompts, validar cada salida y preparar el entregable final.

  1. 1 Elige un modelo Define la profundidad del proyecto antes de iniciar el flujo.
  2. 2 Avanza paso a paso Sigue la secuencia de prompts del modelo activo.
  3. 3 Copia y ejecuta Usa el prompt original en tu herramienta de IA sin modificar su estructura.
  4. 4 Valida la salida Revisa el checklist, marca el estado y confirma si el output puede avanzar.
  5. 5 Prepara el delivery Ordena el resultado final según el objetivo del proyecto o presentación.
